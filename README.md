# Everybody Edits Client Archive
An archive of game clients for the 2010 Adobe Flash game, Everybody Edits.

## How to Contribute
You can [fork](https://github.com/bobatealee/everybody-edits-clients/fork) this repository, add your files and/or correct information, and create a [pull request](https://github.com/bobatealee/everybody-edits-clients/pulls).

### Replacing Unverified Files
Clients labeled as 'unknown' or 'conflicted' could use your help! In most cases, information about these is not known, such as true client versions.

If you think one of these clients could use improving, please create a pull request.

### Sorting Clients
Clients are sorted into folders according to their server version, which can be found inside of the client (usually in the `Config` file).

Clients themselves are named according to their compile date and client version. This can be automated by dragging numerous clients into the `everybody_edits_client_sorter.py` file.

Unknown client versions are labeled as 'unknown', while any clients that are byte-identical are labeled as 'conflicted'.
