# Everybody Edits Client Archive
An archive of game clients for the 2010 Adobe Flash game, Everybody Edits.

## Contribute
Create a [pull request](https://github.com/bobatealee/everybody-edits-clients/pulls) adding your client(s) to the correct paths (e.g. `v204/2015_12_25_v612.swf`)

Client sorting method is as follows:
- Server version (verified in client, usually via Config.as) (e.g. `v204`)
- Compilation date (verified by client metadata) (e.g. `2015_12_25`)
- Client version (Player.IO asset version, if present; else, "unknown") (e.g. `v612`)
